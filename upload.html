hushfile is a file sharing service where the file is encrypted before upload. This enables you to share files while keeping them private from server operators and eavesdroppers. Just pick a file and it will be encrypted in your browser before it is uploaded. When the process is finished you will receive a link which you can share with anyone you wish. Just keep the link secret, it contains the password to decrypt the file!
